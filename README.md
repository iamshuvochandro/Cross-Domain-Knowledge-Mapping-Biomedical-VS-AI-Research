# Cross-Domain-Knowledge-Mapping-Biomedical-VS-AI-Research

This project uses text mining, TF-IDF feature extraction, UMAP dimensionality reduction, and K-means clustering in R to analyze a cross-domain corpus of biomedical and AI research abstracts. It successfully maps domain-specific vocabularies and surfaces overlapping knowledge transfer zones where AI methodologies intersect with healthcare challenges.

You can check Research of this project here - https://github.com/iamshuvochandro/Researches
